ML_test
====================

This project is a machine learning practice testing that uses IMU datas to detect falls.


## 目錄介绍

- **Data** : IMU data

- **Features**
6 axis:X,Y,Z,norm,hori,verti
mean,std,var,max,min,kurtosis,skew,range

- **Model** : Support Vector Machine, SVM
